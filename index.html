05 · Feature

Web API Sidebar

OData Builder
Visually craft $select, $filter, $expand and $orderby with field-browse support.
FetchXML Tester
Write, run and inspect FetchXML with live formatted response and CSV / Excel export.
Structured Filter Builder New
The ⚙ Builder pill opens an inline panel: pick field + operator + value with and/or connectors and optional grouping. Operators include endswith, is null, in, Today, LastXDays, NextXDays and Between. Free-text editing still works.
Rewritten GET → FetchXML New
Lookup $select strips _…_value correctly, nested $expand and filter trees convert properly, and date / null / in operators map to the right FetchXML. New Format & Minify buttons, and OData now emits a full URL.
Snippet Variables & Generate Body New
Snippets substitute {{orgUrl}}, {{tenantId}}, {{envId}}, {{recordId}}, {{entitySetName}} and more at copy time. Generate request body (formerly "Scaffold from fields") is method-aware — POST includes all writable fields; PATCH/PUT drop statecode / statuscode.

07 · Tips

Pro Tips

⌨️
Search Shortcuts

Use fragments like account or av_ — no need to type the full name.

🌍
Environment ID

If "Open in Maker" gives a 404, check the Environment ID in the Solution Picker. Find it in your Maker URL: environments/{THIS-PART}/…

📋
Accurate Logical Names

Use this extension while writing C# Plugins, JavaScript or Power Automate expressions — 100% accurate names, every time.
